# Senior Market Researcher — Common Requirements (2025-2026)

주요 채용 플랫폼의 시니어 마켓 리서처 JD 종합 분석.
indie-market-researcher 스킬 강화를 위한 참고 자료.

---

## 1. Technical Skills (Hard Skills)

| Category | Common Requirements | Notes |
|----------|-------------------|-------|
| **Research Methods** | Primary research (인터뷰, FGI, 설문), Secondary research (보고서, 데이터베이스, 공개자료 분석) | 시니어는 mixed-method 설계 능력 필수 |
| **Survey Design** | 설문 설계, 측정 도구 개발, 샘플링 전략 수립 | Qualtrics, SurveyMonkey, Typeform 숙련 |
| **Data Analysis** | 통계 분석 (회귀, 클러스터링, 요인분석), 정성 데이터 코딩 | SPSS, R, Python (pandas/scipy), Stata |
| **Market Sizing** | TAM/SAM/SOM 산출, Top-down & Bottom-up 추정, 수요 예측 | 투자자 피칭에도 활용되므로 스타트업에서 특히 중시 |
| **Competitive Analysis** | 경쟁사 모니터링, SWOT, Porter's Five Forces, 벤치마킹 | Crayon, Klue, SEMrush 등 CI 자동화 도구 활용 증가 |
| **Data Visualization** | 대시보드 구축, 경영진 대상 리포트 작성 | Tableau, Power BI, Looker, Google Data Studio |
| **Segmentation** | 고객 세분화, 페르소나 개발, 코호트 분석 | AI 기반 micro-cohort 발굴이 2025-2026 트렌드 |
| **Pricing Research** | 가격 탄력성 분석, Conjoint Analysis, Van Westendorp | indie-monetize와 직접 연결되는 역량 |
| **Trend Analysis** | 산업 동향 트래킹, 이머징 기술 모니터링, 시나리오 플래닝 | Google Trends, 특허/논문 DB, 뉴스 모니터링 |
| **Reporting** | 리서치 리포트 작성, Executive Summary, 전략적 권고안 도출 | "So what?" → Actionable Insight 전환 능력 핵심 |

## 2. Senior Differentiators (Mid → Senior Gap)

| Competency | Description | Source |
|-----------|-------------|--------|
| **Strategic Framing** | 리서치 결과를 비즈니스 전략·의사결정에 직접 연결. "무엇을 해야 하는가"까지 제안 | Indeed, LinkedIn JD |
| **Stakeholder Management** | C-level 및 크로스펑셔널 팀 대상 프레젠테이션, 복잡한 분석을 비전문가 언어로 전환 | Glassdoor, O*NET |
| **Research Program Design** | 단일 프로젝트가 아닌 연간 리서치 로드맵 수립, 여러 프로젝트 동시 관리 | LinkedIn Senior JD |
| **Autonomous Research Ownership** | 리서치 계획부터 전략 권고까지 단독 추진. 멘토링/팀 리딩은 팀 환경 한정 | Indeed, ZipRecruiter |
| **Vendor Management** | 외부 리서치 에이전시·패널 업체 활용. 예산 대규모 관리는 Enterprise 한정 — 인디에서는 셀프서브 도구로 대체 가능 | Indeed Senior JD |
| **Business Acumen** | P&L 이해, 매출 영향 분석, ROI 기반 리서치 우선순위 설정 | Radiall, Coursera Guide |
| **Cross-functional Influence** | Product, Marketing, Sales, Biz Dev 팀과 협업하여 리서치 인사이트를 제품 로드맵에 반영 | LinkedIn, Glassdoor |
| **Narrative Storytelling** | 데이터를 "compelling story"로 변환 — 숫자 나열이 아닌 맥락과 함의를 전달 | Indeed, SCIP |

## 3. Soft Skills

| Skill | Detail |
|-------|--------|
| **Analytical Thinking** | 대량 데이터에서 패턴 식별, 가설 수립→검증 사이클 반복 |
| **Communication** | 복잡한 분석을 명확한 언어와 시각자료로 전달 (written + verbal) |
| **Intellectual Curiosity** | 이머징 기술·시장 변화에 대한 지속적 관심과 자기주도 학습 |
| **Critical Thinking** | AI 생성 인사이트를 포함한 모든 데이터의 신뢰성·편향 검증 |
| **Project Management** | 다수 리서치 프로젝트 동시 운영, 타임라인·예산·품질 관리 |
| **Adaptability** | 빠르게 변화하는 시장 환경·도구·방법론에 유연하게 대응 |
| **Persuasion** | 데이터 기반 권고안을 이해관계자에게 설득력 있게 전달 |

## 4. Experience Requirements

| Condition | Typical Range |
|-----------|--------------|
| **총 경력** | 5-8년 (시니어), 8년+ (리드/디렉터) |
| **관련 분야 경력** | Market Research, Consumer Insights, CI, Product Marketing 중 5년+ |
| **학력** | Bachelor's (필수) — Business, Marketing, Economics, Statistics, Data Analytics |
| **석사 우대** | MBA 또는 Data Science/Statistics 석사 — 시니어 이상에서 선호 |
| **자격증 (선호)** | SCIP SIC Certification, ACI CIP, Google Analytics, Tableau Certified |
| **산업 경험** | Tech/SaaS 경험 우대, B2B와 B2C 모두 경험 시 강점 |
| **급여 범위 (US)** | $77K-$186K (Senior), 중앙값 약 $110K-$130K |

## 5. 2025-2026 Trend Keywords

| Trend | Status |
|-------|--------|
| **AI-Augmented Research** | 설문 설계·코딩·분석 자동화 — 수주 걸리던 작업이 수시간으로 단축 |
| **Competitive Intelligence Automation** | Crayon, Klue 등으로 경쟁사 가격·채용·제품 변경 실시간 추적 + AI SWOT 자동 생성 |
| **Advanced NLP Sentiment Analysis** | 단순 긍/부정을 넘어 sarcasm, urgency, brand fatigue 감지 |
| **AI-Free Critical Thinking** | [SPECULATIVE] Gartner 예측: 2026년까지 글로벌 기업 50%가 "AI 없는" 비판적 사고력 평가 도입 (아직 채용 요구사항에 명시된 사례 드묾) |
| **Micro-Cohort Discovery** | 비지도 학습 기반 세분화 — 전통적 인구통계를 넘어선 초정밀 고객군 발굴 |
| **Product-Led Growth Research** | PLG 모델에서의 사용자 행동 분석, Activation·Retention 메트릭 기반 리서치 수요 증가 |
| **Real-time Market Intelligence** | 정적 보고서 → 실시간 대시보드 전환 가속 |
