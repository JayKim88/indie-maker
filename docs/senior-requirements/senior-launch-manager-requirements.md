# Product Launch Manager — Common Requirements (2025-2026)

주요 채용 플랫폼의 Product Launch Manager / Go-to-Market Manager JD 종합 분석.
indie-launcher 스킬 강화를 위한 참고 자료.

---

## 1. Technical Skills (Hard Skills)

| Category | Common Requirements | Notes |
|----------|-------------------|-------|
| **Go-to-Market Strategy** | GTM 플랜 수립, 포지셔닝/메시징 프레임워크, 런치 타임라인 설계 | 거의 모든 JD에서 핵심 요구사항 |
| **Product Marketing** | 제품 포지셔닝, 경쟁 분석, buyer persona 개발, 마케팅 collateral 제작 | PMM 역할과 높은 중복 |
| **Data Analysis & ROI Tracking** | 캠페인 성과 측정, 전환율 분석, A/B 테스트, attribution 모델링 | "Exceptional ROI-tracking skills" 빈출 |
| **Project Management** | 크로스-펑셔널 프로젝트 리드, 런치 체크리스트 관리, 타임라인/마일스톤 추적 | PMP/Agile 인증 선호 |
| **Marketing Automation & Tools** | HubSpot, Marketo, Salesforce, Google Analytics, Mixpanel | 툴 스택 경험 필수 |
| **Content & Messaging** | 프레스 릴리스, 랜딩페이지 카피, 이메일 시퀀스, 소셜 미디어 전략 | 런치 collateral 전반 |
| **Market Research** | TAM/SAM/SOM 분석, 고객 인터뷰, 경쟁사 벤치마킹, 트렌드 분석 | 데이터 기반 의사결정 강조 |
| **Sales Enablement** | 세일즈 덱, 데모 스크립트, 온보딩 자료 제작. B2B SaaS 팀 환경 한정 — 인디에서는 랜딩페이지·이메일 시퀀스로 대체 가능 | B2B SaaS에서 특히 중요 |

## 2. Senior Differentiators (Mid → Senior Gap)

| Competency | Description | Source |
|-----------|-------------|--------|
| **Strategic GTM Ownership** | 단일 제품이 아닌 포트폴리오/플랫폼 수준의 GTM 전략 수립 및 P&L 책임 | LinkedIn, Glassdoor |
| **Cross-functional Leadership** | 엔지니어링, 세일즈, CS, 디자인 팀을 formal authority 없이 이끄는 능력 | Indeed |
| **Executive Communication** | C-suite 대상 런치 성과 리포팅, 보드 프레젠테이션, 전략적 내러티브 구성 | Wellfound |
| **PLG Integration** | Product-Led Growth 전략과 런치를 통합, self-serve 퍼널 설계 | 2025-2026 트렌드 |
| **AI-Powered Launch Ops** | AI 툴 활용한 콘텐츠 생성, 타겟 세분화, 캠페인 자동화 | 2026 JD에서 급증 |
| **Multi-channel Orchestration** | Product Hunt, Hacker News, 커뮤니티, PR, 파트너십 동시 조율 | 인디/스타트업 특화 |
| **Post-launch Analytics** | 런치 후 retention/activation 분석, 피드백 루프 구축, iteration 로드맵 수립 | Senior PMM 차별화 |

## 3. Soft Skills

| Skill | Detail |
|-------|--------|
| **Storytelling & Narrative** | 제품 가치를 고객 관점의 스토리로 전환, 채널별 톤 조절 능력 |
| **Stakeholder Management** | 경영진, 엔지니어링, 디자인, 세일즈 등 다양한 이해관계자 조율 및 기대치 관리 |
| **Empathy & User Advocacy** | 고객 니즈와 pain point에 대한 깊은 이해, 사용자 중심 런치 설계 |
| **Adaptability Under Pressure** | 런치 일정 변경, 예상치 못한 이슈 대응, 빠른 의사결정 |
| **Influence Without Authority** | 직접 보고 라인 없는 팀원들을 동기부여하고 방향 정렬 |

## 4. Experience Requirements

| Condition | Typical Range |
|-----------|--------------|
| **Mid-level (Manager)** | 3-5년, 최소 2회 이상 full product launch 경험 |
| **Senior (Sr. Manager / Director)** | 5-10년, B2B SaaS 또는 테크 산업 경험 필수 |
| **학위** | Bachelor's (Marketing, Business, Engineering), MBA 선호 |
| **연봉 범위 (US)** | $90K-$150K+ (레벨/지역에 따라 상이) |

## 5. 2025-2026 Trend Keywords

| Trend | Status |
|-------|--------|
| **AI-Powered Content & Campaign** | 급부상 — AI 활용 콘텐츠 생성, 타겟팅 자동화가 기본 역량 |
| **Product-Led Growth (PLG)** | 주류 — 런치 전략이 self-serve 온보딩/activation과 통합 |
| **Community-Led Launch** | 성장 — Discord, Reddit, 인디해커 커뮤니티 중심 런치 전략 증가 |
| **Data-Driven Launch Metrics** | 필수 — Vanity metric 대신 activation rate, time-to-value 등 실질 지표 |
| **Cross-functional GTM Pods** | 확산 — 마케팅-세일즈-프로덕트-CS 통합 소규모 팀 구조 |

## 6. Role Boundary Note

**Launch vs Growth**: Launch Manager는 D-7~D+7 출시 윈도우에 집중 — 채널 스택킹, 초기 모멘텀, 프레스. Growth는 출시 이후 지속적 획득·리텐션·수익화 최적화 담당. 인디 환경에서 한 명이 겸임하되, 시간 축 기준으로 역할을 구분하는 것이 효과적.
