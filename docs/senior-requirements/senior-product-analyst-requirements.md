# Senior Product/Data Analyst — Common Requirements (2025-2026)

주요 채용 플랫폼의 시니어 프로덕트/데이터 분석가 JD 종합 분석.
indie-analyst 스킬 강화를 위한 참고 자료.

---

## 1. Technical Skills (Hard Skills)

| Category | Common Requirements | Notes |
|----------|-------------------|-------|
| **SQL / Database** | SQL (advanced joins, window functions, CTEs), Snowflake, BigQuery, Redshift, Postgres | 거의 모든 JD에서 필수. "Expert-level SQL" 표현 빈번 |
| **Programming** | Python, R (통계 분석·커스텀 모델링 용도) | Python이 기본 언어로 자리잡음. R은 선택적 |
| **Data Visualization / BI** | Tableau, Looker, Power BI, Mode, Sigma | 셀프서비스 대시보드 구축·유지 역량 요구 |
| **Product Analytics Tools** | Amplitude, Mixpanel, Heap, PostHog, Pendo, FullStory | SaaS/테크 기업에서 필수. 행동 분석·퍼널·코호트 분석 |
| **A/B Testing / Experimentation** | 실험 설계, t-test, confidence intervals, uplift calculation, 통계적 유의성 판단 | 프로덕션 환경에서의 A/B 테스트 프레임워크 운영 경험 강조 |
| **Statistical Analysis** | 가설 검정, 회귀 분석, causal inference, 예측 모델링 | "statistical rigor" 키워드 반복 등장 |
| **Product Metrics** | Retention, activation, conversion, LTV, churn, funnel analysis, cohort analysis | 메트릭 정의·측정 체계 수립 경험 중시 |
| **Data Modeling / Transformation** | dbt, data warehousing concepts, ETL/ELT pipeline 이해 | Modern Data Stack 경험 우대 |
| **Web Analytics** | Google Analytics (GA4), event tracking, user journey mapping | PLG(Product-Led Growth) 기업에서 특히 중시 |
| **GenAI / AI Literacy** | LLM 활용, prompt engineering, AI-assisted analytics | 2025년 이후 급격히 수요 증가 |

## 2. Senior Differentiators (Mid → Senior Gap)

| Competency | Description | Source |
|-----------|-------------|--------|
| **분석 → 전략 전환** | 분석 실행자에서 전략 조언자로 전환. 무엇을 만들지, 왜 만드는지에 영향을 미침 | Product School, Userpilot |
| **메트릭 프레임워크 수립** | 팀이 사용할 KPI·OKR 체계를 직접 정의하고, 측정 기준을 조직 전체에 배포 | Toast, Bloomerang JD |
| **실험 문화 리딩** | A/B 테스트 프레임워크를 단독 설계·운영. 통계적 오류 방지 프로세스 구축 | Glassdoor, Indeed JD |
| **크로스펑셔널 영향력** | PM, 엔지니어링, 디자인, 경영진과 직접 협업. 데이터로 의사결정 방향을 전환시킨 경험 | Multiple JDs |
| **자율적 실행 (Autonomous Ownership)** | 팀 지원 없이 분석 프로젝트 전체를 리드. 멘토링은 팀 환경 한정; 인디/소규모 컨텍스트에서는 독립 실행 역량이 핵심 | data.org, Velvet Jobs |
| **데이터 스토리텔링** | 복잡한 데이터를 경영진 수준의 내러티브로 변환. "신뢰받는 목소리"로서의 위치 | Teal HQ, Medium |
| **분석 로드맵 관리** | 여러 프로덕트 팀의 분석 프로젝트를 우선순위화·관리. 리소스 배분 의사결정 | Rover, Toast JD |
| **질문을 묻는 역할** | 답을 제공하는 것을 넘어, 올바른 질문을 조직에 제기하는 역할 | Product School |

## 3. Soft Skills

| Skill | Detail |
|-------|--------|
| **Data Storytelling** | 데이터 분석 결과를 설득력 있는 내러티브로 구성. 시작-중간-결론 구조로 이해관계자에게 전달 |
| **Stakeholder Management** | 비기술 이해관계자와 기술 팀 사이 브릿지 역할. 데이터 기반 변화를 옹호하는 능력 |
| **Communication & Presentation** | 경영진·이사회 대상 프레젠테이션. 복잡한 정보를 명확하고 효과적으로 전달 |
| **Strategic Thinking** | 비즈니스 컨텍스트 이해 기반의 전략적 파트너링. 분석 결과를 비즈니스 목표에 연결 |
| **Problem Framing** | 모호한 비즈니스 문제를 분석 가능한 질문으로 구조화 |
| **Cross-functional Collaboration** | PM, 엔지니어, 디자이너, 마케팅 등 다양한 팀과 효과적으로 협업 |
| **Intellectual Curiosity** | 스스로 질문을 발굴하고, 데이터의 이면을 파고드는 자기주도적 탐구심 |
| **Attention to Detail** | 데이터 정확성·무결성 보장. 분석 결과의 신뢰도를 유지하는 꼼꼼함 |

## 4. Experience Requirements

| Condition | Typical Range |
|-----------|--------------|
| **총 경력 연수** | 5-7년 (데이터 분석 또는 관련 분야) |
| **최소 경력** | 3-5년 (일부 JD 기준) |
| **학력** | Bachelor's Degree (필수) — Statistics, CS, Economics, Mathematics, Business 등 |
| **학력 우대** | Master's Degree — 더 전문적 역할·높은 보상에 유리 |
| **도메인 경험** | SaaS, fintech, e-commerce, marketplace 등 특정 도메인 경험 우대 |
| **연봉 (US, 2025)** | $88K-$155K (25th-75th percentile), 평균 ~$126K, 상위 10% $161K+ |
| **인증 (선택)** | Google Data Analytics, dbt Analytics Engineering, Tableau Certified 등 |

## 5. 2025-2026 Trend Keywords

| Trend | Status |
|-------|--------|
| **GenAI / LLM Literacy** | 필수화 진행 중. 2021년 0건 → 2025년 ~10,000건 이상의 GenAI 관련 채용 공고 |
| **Prompt Engineering** | 프롬프트 엔지니어링이 분석가에게도 기본 스킬로 요구되기 시작 |
| **AI-Assisted Analytics** | 파이프라인 생성, 이상 탐지, 자동 요약 등 스택 전 계층에 AI 에이전트 도입 |
| **Product-Led Growth Analytics** | 셀프서비스 전환 퍼널, activation metric, feature adoption 분석 역량 수요 급증 |
| **dbt + Modern Data Stack** | dbt 기반 transformation이 분석가 워크플로우의 표준으로 정착 |
| **Causal Inference** | 단순 상관관계를 넘어 인과관계 추론 역량 중시 (A/B 테스트 한계 보완) |
| **Self-Serve Data Culture** | 분석가가 대시보드·데이터 모델을 구축해 비분석 팀이 스스로 답을 찾도록 지원 |
| **Experimentation Platform** | 자체 실험 플랫폼 구축·운영 경험이 차별화 요소로 부상 |
