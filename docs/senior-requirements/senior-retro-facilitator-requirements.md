# Retrospective / Post-mortem Facilitator — Common Requirements (2025-2026)

주요 채용 플랫폼의 Agile Coach / Continuous Improvement Manager JD 종합 분석.
indie-retro 스킬 강화를 위한 참고 자료.

---

## 1. Technical Skills (Hard Skills)

| Category | Common Requirements | Notes |
|----------|-------------------|-------|
| **Agile Frameworks** | Scrum, Kanban, XP — 프레임워크별 retrospective 형식 이해. SAFe/LeSS는 Enterprise 한정 — 인디 환경에서는 불필요 | Scrum이 가장 빈출 |
| **Retrospective Facilitation** | Start/Stop/Continue, 4Ls, Sailboat, Mad/Sad/Glad 등 다양한 retro 포맷 운영 | 상황별 포맷 선택 능력이 핵심 |
| **Blameless Post-mortem** | Google SRE 스타일 blameless postmortem 설계/진행, incident timeline 구성 | DevOps/SRE 팀에서 특히 중요 |
| **Root Cause Analysis** | 5 Whys, Fishbone Diagram, Fault Tree Analysis | 문제의 근본 원인 도출 역량 |
| **Lean & Six Sigma** | Kaizen 이벤트, DMAIC, Value Stream Mapping, 낭비 제거 | Continuous Improvement Manager 계열 |
| **Metrics & Measurement** | Velocity, cycle time, lead time, DORA metrics, team health check | 개선 효과를 정량적으로 증명 |
| **Tools** | Jira, Confluence, Miro, FigJam, Parabol, TeamRetro, Notion | 비동기/원격 retro 도구 운영 |
| **Knowledge Management** | Lessons learned DB 구축, action item 추적, 조직 지식 체계화 | Retro 결과의 조직적 활용 |

## 2. Senior Differentiators (Mid → Senior Gap)

| Competency | Description | Source |
|-----------|-------------|--------|
| **Continuous Improvement Culture** | 팀 단위의 continuous improvement 습관 구축. 조직 전체 변환은 Enterprise Agile Coach 한정 — 인디에서는 스프린트 학습 루프 정착이 핵심 | LinkedIn Enterprise Agile Coach JD |
| **Systemic Pattern Recognition** | 개별 retro에서 반복 패턴 식별, 조직 수준의 시스템적 문제 진단 | Scrum Alliance Coaching Competency |
| **Psychological Safety Building** | 심리적 안전감 조성을 위한 체계적 접근, 팀 신뢰도 측정 및 개선 | Google re:Work 기반 JD |
| **Executive Coaching** | 경영진 대상 코칭. Enterprise/Director+ 한정 — 인디 환경에서는 팀 내 심리적 안전감 조성이 동등한 역할 | ICP-ACC 역량 모델 |
| **Multi-team Coordination** | 팀 간 학습 전파 체계. Program/Portfolio 수준 retro는 Enterprise/SAFe 한정 — 인디에서는 lessons.md 기반 다음 스프린트 반영이 핵심 | SAFe SPC JD |
| **Cultural Diagnostics** | 조직 문화 진단 도구 활용, 변화 저항 분석, 전환 로드맵 수립 | Enterprise Transformation Lead |
| **ROI of Improvement** | 개선 활동의 비즈니스 임팩트를 정량적으로 측정하고 경영진에 보고 | Continuous Improvement Director JD |

## 3. Soft Skills

| Skill | Detail |
|-------|--------|
| **Active Listening** | 팀원의 표면적 발언 너머의 감정과 우려를 파악, 안전한 공간 조성 |
| **Facilitation & Neutrality** | 특정 입장에 치우치지 않는 중립적 진행, 모든 목소리가 들리도록 보장 |
| **Conflict Resolution** | 비난 없는 토론 유도, 갈등 상황에서 생산적 대화로 전환 |
| **Empathy & Emotional Intelligence** | 팀 역학과 개인 감정 상태에 대한 민감한 인식 |
| **Coaching Mindset** | 답을 주지 않고 질문으로 통찰을 이끌어내는 접근 |
| **Influence Without Authority** | 직접 권한 없이 팀과 리더십의 행동 변화를 이끄는 능력 |
| **Storytelling for Learning** | 실패와 성공 사례를 조직 학습에 활용할 수 있는 내러티브로 구성 |

## 4. Experience Requirements

| Condition | Typical Range |
|-----------|--------------|
| **Mid-level (Agile Coach)** | 5-7년, Scrum Master + 코칭 경험, ICP-ACC 또는 동급 인증 |
| **Senior (Enterprise Coach / CI Director)** | 8-12+년, 조직 변환 프로젝트 리드, SAFe SPC 또는 CEC |
| **학위** | Bachelor's (CS, Engineering, Business), Master's 선호 |
| **인증 (빈출 순)** | CSM → PSM II → ICP-ACC → ICP-ATF. SAFe SPC/CEC는 Enterprise 한정 |
| **연봉 범위 (US)** | Scrum Master $90K-$120K / Agile Coach $130K-$170K / Enterprise $180K-$250K+ |

## 5. 2025-2026 Trend Keywords

| Trend | Status |
|-------|--------|
| **Async Retrospectives** | 정착 — 원격/분산 팀 환경에서 비동기 retro 도구 활용 일상화 |
| **Continuous Retrospectives** | 성장 — 스프린트 끝이 아닌 진행 중 수시 피드백 수집 (always-on retro) |
| **Blameless Culture at Scale** | 확산 — SRE 커뮤니티 너머 일반 프로덕트 팀으로 blameless postmortem 확산 |
| **AI-Assisted Facilitation** | 초기 — AI가 retro 데이터 요약, 패턴 식별, action item 추적 지원 |
| **Psychological Safety Metrics** | 주류 — Google re:Work 기반 팀 건강 측정이 표준 프랙티스 |
| **Learning Organization Framework** | 성장 — 개별 retro를 넘어 조직 학습 체계(knowledge base, lessons DB) 구축 |
| **Kill Decision Frameworks** | 니치 — 인디/스타트업 환경에서 프로젝트 중단 의사결정을 위한 구조화된 retro |

## 6. Indie Context Note

**인디/소규모 팀에서의 retro 우선순위**: Blameless Post-mortem + Kill Decision Framework + lessons.md 구축이 핵심. SAFe/LeSS/Portfolio Retro 등 Enterprise 패턴은 스킬 범위 밖. 심리적 안전감과 빠른 학습 루프가 조직 변환 프로그램을 대체.
