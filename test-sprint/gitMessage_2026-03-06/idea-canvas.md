## One-Line Definition
GitMessage is a VS Code extension for solo developers to generate conventional commit messages instantly from git diff.

## Problem
**Target**: Solo developers and indie hackers using VS Code or Cursor
**Situation**: After finishing a feature, right before committing
**Problem**: Writing good commit messages is friction — results in lazy messages like "fix stuff" or "update"
**Current solution**: Manual typing, or aicommits CLI (requires terminal context switch)

## Differentiation
Why I'm well-positioned to solve this:
- VS Code extension development experience
- I am the target user — exact UX need is clear
- Key insight: Source Control panel integration (no terminal switch) is the gap nobody has filled
