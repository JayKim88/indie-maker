# Design Brief: GitMessage

> Created: 2026-03-06 | Phase 2 Design Sprint

---

## UX Architecture Constraints (from ux-flow.md — priority over design style)

- Navigation: VS Code native only (SCM panel + status bar + command palette)
- No sidebar views or activity bar icons in MVP
- Onboarding: Zero-friction (install → button appears, no welcome screen)

---

## Brand Identity

**Product type**: Dev Tool
**Brand feel**: Minimal / Neutral + AI signal
**Personality**: Precise, invisible, fast — "it just works"

### Color System

| Token | Value | Usage |
|-------|-------|-------|
| Primary | `#7C3AED` | Buttons, ✨ icon, active state |
| Background | `#1E1E1E` | Webview background (matches VS Code dark) |
| Surface | `#252526` | Card / panel backgrounds |
| Text Primary | `#CCCCCC` | Body text |
| Text Muted | `#6B7280` | Secondary text, placeholder |
| Success | `#4EC9B0` | Confirmed / committed state |
| Danger | `#F44747` | Error states |
| Border | `#3E3E42` | Dividers |

### Typography

- **Code / monospace**: JetBrains Mono (commit messages, diff previews)
- **UI**: `-apple-system, "Segoe UI", sans-serif` (VS Code native system font)
- **Size scale**: 11px (caption), 13px (body), 15px (label), 20px (heading)

### Iconography

- ✨ Sparkle: primary generate action (matches VS Code Copilot language)
- Lucide icons for all other UI elements
- No custom icon set needed for MVP
